# Огляд API

Цей документ містить приклади API-запитів для онлайн-магазину книжок.

## GET /api/books
Отримати список всіх книжок.

**Приклад запиту:**
GET /api/books
**Приклад відповіді:**
ID: 1
Назва: Кобзар
Автор: Тарас Шевченко
Ціна: 250 грн

ID: 2
Назва: Лісова пісня
Автор: Леся Українка
Ціна: 200 грн

## POST /api/orders
Створити нове замовлення користувача.

**Приклад запиту:**
POST /api/orders
Користувач: ID 5
Книги: 1, 3
**Приклад відповіді:**
Статус: Замовлення створено успішно
Номер замовлення: 1024

## Коди відповідей

| Код | Опис |
|----|------|
| 200 | Запит виконано успішно |
| 400 | Некоректний запит |
| 404 | Ресурс не знайдено |
| 500 | Внутрішня помилка сервера |
