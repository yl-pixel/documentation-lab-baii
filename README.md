# Онлайн-магазин книжок

Онлайн-магазин книжок — це вебсистема для перегляду, пошуку та купівлі книжок через інтернет.

## Проблема, яку вирішує система
Користувачам часто незручно шукати потрібні книги в офлайн-магазинах. Цей сервіс дозволяє швидко знайти книгу, переглянути опис і зробити замовлення онлайн.

## Основні функції
- Перегляд каталогу книжок
- Пошук книжок за назвою або автором
- Перегляд детальної інформації про книгу
- Додавання книжок у кошик
- Оформлення замовлення

## Для кого цей продукт
- Студенти
- Любителі читання
- Покупці, які хочуть замовляти книги онлайн
